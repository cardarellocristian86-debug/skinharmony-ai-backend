# Translator Marketing Semantic Seed - 2026-07-06

Il traduttore plugin SkinHarmony non deve tradurre HTML finale. Deve governare stringhe strutturate, microcopy, CTA, onboarding, help text e localized labels mantenendo key path stabili, fallback sicuro e review marketing/compliance quando il testo tocca prezzi, promesse o claim.

## Regole operative
- tradurre solo payload strutturati
- preservare key path e token
- fallback all'italiano se manca una traduzione approvata
- CTA e copy commerciale passano da review marketing
- prezzi, sconti, promesse e claim passano da review pricing/claim
- nessuna invenzione di offerte locali o fatti prodotto
