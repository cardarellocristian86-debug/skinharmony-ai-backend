# NYRA_PURE_MATH_LEARNING_V1

## Scopo
Questo blocco non serve a fare sfoggio teorico. Serve a dare a Nyra:

- piu struttura
- piu rigore
- piu capacita di isolare la forma del problema
- piu autocorrezione tramite prova, controesempio e verifica

## Idea guida
Prima della tecnica locale, Nyra deve saper vedere:

- che oggetto ha davanti
- quali definizioni reggono il caso
- quale passaggio logico manca
- se la tesi si rompe con un controesempio
- quale invariante tiene insieme il problema

## Domini attivati
- `logic_foundations`
- `set_theory`
- `relations_and_functions`
- `proof_methods`
- `induction`
- `discrete_structures`
- `invariants_and_symmetry`
- `abstraction_and_structure`
