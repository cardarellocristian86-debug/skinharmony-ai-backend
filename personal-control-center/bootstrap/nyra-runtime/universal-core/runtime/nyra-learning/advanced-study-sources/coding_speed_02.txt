Cargo downloads your Rust package ’s dependencies, compiles your packages, makes distributable packages, and uploads them to crates.io , the Rust community’s package registry . You can contribute to this book on GitHub . Sections Getting Started To get started with Cargo, install Cargo (and Rust) and set up your first crate .
